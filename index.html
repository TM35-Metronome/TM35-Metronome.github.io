TM35-Metronome

Tools | FQA

TM35-Metronome (or Metronome for short) is a collection terminal based tools for modifying Pokémon games. Unlike existing tools such as PPRE, BWWE, and many others, Metronome aims to work on any generation of Pokémon games and provides a simple interface that allows extension in any programming language.

Example

Let's pretend that you want to change the typing of the Pokémon Bulbasaur in Pokémon Fire Red. First, you'll need one of the tm35-genX-load programs to extract game information into an editable text file.

tm35-gen3-load pokemon-fire-red.gba > fire-red.tm35

Inside this file, you'll find a lot of information about the game you loaded. What we are looking for is Bulbasaur's information.

Metronome does not support extracting Pokémon names yet. You, therefore, cannot search for Bulbasaur. The best way to find the Pokémon you're looking for is searching for it using its index number. Bulbasaur's index number is 1 so let's search for .pokemons[1].types.
.pokemons[1].types[0]=Grass
.pokemons[1].types[1]=Poison

Let's change its second typing to Dragon.

.pokemons[1].types[0]=Grass
.pokemons[1].types[1]=Dragon

If we now save this file and pipe it into tm35-gen3-apply, our changes will be applied to the game.

tm35-gen3-apply pokemon-fire-red.gba -o fire-red-modified.gba < fire-red.tm35

If you already know what you want to modify you can also use tm35-genX-apply on its own.

echo ".pokemons[1].types[1]=Dragon" | tm35-gen3-apply pokemon-fire-red.gba -o fire-red-modified.gba

Metronome also provides a set of tools that can read, understand and modify this data. One of these tools is tm35-rand-wild which will randomize the wild Pokémons in a game.

tm35-gen3-load pokemon-fire-red.gba | tm35-rand-wild | tm35-gen3-apply pokemon-fire-red.gba -o fire-red-modified.gba
